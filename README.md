# Simply Chocolate

Simply Chocolate is a high-performance, **adaptive landing page** designed for a
seamless user experience across smartphones, tablets, and desktops.

The codebase adheres strictly to **HTML5** and **CSS3** standards, prioritizing
**semantic markup** and passing code validation with zero errors or warnings.
Stylesheets organized by the **Concentric CSS** methodology, ensuring the site
is not only visually appealing but also scalable and accessible. By leveraging
**CSS Custom Properties** (variables) for design tokens and
[modern-normalize](https://cdnjs.com/libraries/modern-normalize), the project
maintains a consistent visual identity and flawless cross-browser performance.

Beyond clean code, the site is fine-tuned for **search engine visibility** and
**screen-reader compatibility**. It is also optimized for **Retina displays**
with high-resolution, minified imagery.

## Tech Stack & Tools

- **Core:** HTML5 (semantic markup), CSS3 (modern layouts), JavaScript
- **Styling & Architecture:** CSS Variables for centralized theme management,
  Concentric CSS for property organization
- **Consistency:**
  [modern-normalize](https://cdnjs.com/libraries/modern-normalize) for
  cross-browser stability
- **Strategy:** Mobile-First Design, Adaptive Web Design (breakpoints: 375px,
  768px, and 1200px)
- **Build System:** Vite
  ([Vanilla App Template](https://github.com/goitacademy/vanilla-app-template))
- **Optimization:** Image minification ([Squoosh](https://squoosh.app/)),
  high-density imagery Retina support, SEO best practices, Web Accessibility
- **Deployment & Validation:** GitHub Pages,
  [W3C Markup Validation Service](https://www.w3.org/developers/tools/)
